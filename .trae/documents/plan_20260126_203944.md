## MachineID-Manage 项目初始化计划

### 核心技术栈

* **Rust + Tauri 2**（后端框架）

* **winreg** crate（Windows 注册表操作）

* **Beads**（任务追踪）

### 任务执行步骤

1. **创建 Rust 工作空间**

   * 初始化 Cargo.toml 工作空间

   * 创建 src-tauri 子项目目录

2. **配置 Tauri 2 依赖**

   * 添加 tauri = "2.9.5"

   * 添加 winreg = "0.50"（注册表操作）

   * 配置必要的 Cargo features

3. **生成项目规则文件** (AGENTS.md)

   * 技术栈约束

   * 开发规范和 TDD 要求

   * Git 提交约定

4. **创建法律声明文件**

   * LICENSE（MIT/Apache 2.0）

   * DISCLAIMER.md（注册表操作风险提示）

   * CONTRIBUTING.md

   * SECURITY.md

5. **构建脚手架**

   * 创建 tauri.conf.json 配置文件

   * 创建基本的 src-tauri/src 结构

   * 创建前端目录占位符

   * 创建 .gitignore

6. **初始化 Beads 追踪**

   * 运行 `bd init`

   * 配置初始任务列表

### ⚠️ 当前阶段禁止

* 实现机器码读取/备份/替换/生成逻辑

* 实现任何功能代码

* 实现前端交互逻辑
